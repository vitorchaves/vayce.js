About Vayce.js

We strongly recommend that you use an ID or class to avoid conflict with other elements in your project.

After making your markup structure it's time to start Vayce. You simply need to reference the menu you want to make responsive and call Vayce. He will do the heavy lifting!


! Note that to create submenus just put the class 'sub-me' in the LI you want to make parent of the submenu.


        
          $('#my-menu').vayce();

Voilá! Vayce has just created a new responsive menu for you! You can check how to customize it and leave it with your face below.


Settings

Check out the list of options you can use on your Vayce:



        
          $('#my-menu').vayce({
            bgColor: "#34495e", // Change the background color of the menu
            fgColor: "#eee", // Change the foreground color (li color) of the menu
            alignment: 'right', // Change the position of the menu
            useDivider: true, // Boolean - Use borders to divide the items or no
            dividerColor: "#546f8a", // Change the divider color
            useImages: true, // Boolean - To allow or disallow image use
            imageWidth: '15px', // Change the image width
            menuColor: "gray", // Change the menu icon color
            menuActive: "#e67e22", // Change the menu icon active color
            topSpace: '40px', // Control the distance the menu should appear in relation the menu icon
            itemsPadding: '9px 25px', // Control the padding into the LI items
            controller: '▼', // Change the icon (Use '' to use nothing :P)
            borderRadius: '5px', // The radius of content menu border
            fullMenu: false, // If you want to use a full menu, just put this to TRUE and be sure that your menu is inside a relative parent.
            menuWidth: "250px" // If diferent of 250px, it will change the menu width, min-width and max-width.
          });


! If you use nothing, this settings will be the default settings.

Full Width Menu Working (Look in small screens. This is inside a container.)

You can choose your container size using CSS. Vayce will respect it.
